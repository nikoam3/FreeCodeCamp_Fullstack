Introducción

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general, conocido por su simplicidad y legibilidad.

Es ampliamente utilizado en desarrollo web, ciencia de datos, automatización y más.

print("¡Hola, mundo!")

  • Gran comunidad
Sintaxis Básica

La sintaxis de Python es clara y utiliza indentación para definir bloques de código.

Los comentarios se crean con el símbolo #.

# Esto es un comentario print("Ejemplo de código")
Variables y Tipos

En Python, no necesitas declarar el tipo de variable explícitamente.

Los tipos principales incluyen enteros, flotantes, cadenas y booleanos.

x = 10 # Entero y = "Hola" # Cadena
  • int: Números enteros
  • str: Cadenas de texto
Estructuras de Control

Python soporta estructuras como if, for y while.

Un ejemplo de un bucle for:

for i in range(5): print(i)

Las condiciones if permiten tomar decisiones basadas en lógica.

Funciones

Las funciones en Python se definen con la palabra clave def.

Un ejemplo de función simple:

def saludar(nombre): return f"Hola, {nombre}"

Puedes llamar a la función con diferentes argumentos.
